# 방셀 헬퍼 사용 가이드

## 개요

방셀 헬퍼는 SOOP 스트리머를 위한 도구로, 다음 기능을 제공합니다:

1. **모바일 → PC 사진 전송**: 휴대폰으로 찍은 셀카를 PC로 바로 전송
2. **자동 저장**: 방셀 작성 중 자동으로 내용 백업
3. **복구 기능**: 브라우저가 꺼져도 작성 중이던 내용 복구

---

## 1. 사진 전송 (모바일 → PC)

### PC에서

1. Chrome 주소창 오른쪽의 **방셀 헬퍼 아이콘** 클릭
2. QR 코드와 연결 코드가 표시됨

### 모바일에서

1. 방셀 헬퍼 PWA 웹페이지 접속
2. 두 가지 연결 방법 중 선택:
   - **QR 코드 스캔**: PC 화면의 QR 코드를 스캔
   - **코드 입력**: 연결 코드 직접 입력 (예: `ABC-123`)

### 연결 후

1. **사진 촬영** 또는 **갤러리에서 선택**
2. 원하는 사진 선택 (최대 10장)
3. **PC로 전송하기** 버튼 클릭
4. 전송 완료 후 PC에서 사진 확인

---

## 2. 자동 저장

### 자동 저장 작동 방식

- SOOP 게시판 글쓰기 페이지에서 자동 활성화
- 글을 작성하면 **0.5초마다** 자동 저장
- 화면 오른쪽 하단에 "✓ 자동 저장됨" 표시

### 저장되는 내용

- 제목
- 본문 내용
- 수신된 사진

### 저장 데이터 확인

- 확장프로그램 팝업에서 "저장된 방셀 초안" 목록 확인
- 초안 클릭 시 해당 페이지로 이동

---

## 3. 초안 복구

### 복구 상황

다음 경우에 초안이 복구됩니다:

- 브라우저를 껐다 다시 켤 때
- 같은 게시판 글쓰기 페이지를 다시 열 때
- PC가 갑자기 꺼졌을 때

### 복구 방법

1. 이전에 작성하던 게시판 페이지 접속
2. "저장된 방셀 초안이 있습니다" 팝업 표시
3. 두 가지 선택:
   - **이어서 작성하기**: 저장된 내용 복원
   - **새로 시작**: 저장된 내용 삭제하고 처음부터

---

## 4. 수신된 사진 사용

### 사진 목록

- 모바일에서 전송한 사진은 에디터 상단에 표시
- 썸네일 미리보기 제공

### 사진 삽입

1. 원하는 사진의 **➕ 버튼** 클릭
2. 에디터 커서 위치에 사진 삽입

### 사진 삭제

1. 원하는 사진의 **🗑️ 버튼** 클릭
2. 개별 삭제

### 모두 삭제

1. 사진 목록 상단의 **"모두 삭제"** 버튼 클릭
2. 확인 후 삭제

---

## 5. 연결 해제

### PC에서

- 팝업의 **"연결 해제"** 버튼 클릭
- 또는 팝업 닫기

### 모바일에서

- **"연결 해제"** 버튼 클릭
- 또는 브라우저 종료

## 6. FAQ

### Q: 사진이 전송되지 않아요

**A**: 다음을 확인하세요:
- PC와 모바일이 같은 연결 코드를 사용 중인지
- 인터넷 연결이 정상인지
- 방화벽이 WebRTC를 차단하지 않는지

### Q: 저장된 초안이 안 보여요

**A**:
- 확장프로그램이 설치되어 있는지 확인
- SOOP 사이트에서 확장프로그램 권한이 허용되어 있는지 확인
- 7일 이상 된 초안은 자동 삭제됨

### Q: 모바일에서 카메라가 작동하지 않아요

**A**:
- HTTPS 환경인지 확인 (http://는 카메라 사용 불가)
- 브라우저 권한 설정에서 카메라 허용했는지 확인

### Q: QR 코드 스캔이 안 돼요

**A**:
- 일부 브라우저는 QR 스캔 미지원
- 연결 코드를 직접 입력하세요

## 7. 개인정보 및 보안

### 데이터 전송

- 사진은 **P2P로 직접 전송**됩니다 (서버 경유 X)
- 시그널링 데이터만 Firebase를 거침 (연결 설정용)
- WebRTC는 기본적으로 **DTLS 암호화** 적용

### 로컬 저장

- 모든 초안과 사진은 **브라우저 로컬(IndexedDB)**에 저장
- 외부 서버로 전송되지 않음
- 7일 후 자동 삭제

### 데이터 삭제

- 확장프로그램 제거 시 모든 데이터 삭제
- 또는 Chrome 설정 → 인터넷 사용 기록 삭제
